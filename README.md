## Tempest atari

### run game
```
mkdir build/ && cd build/
cmake ..
make

# run

./bin/Tempest

```
or

```
./run.sh
./bin/Tempest
```
### Test game

```
./run.sh test

```

### clean game
```
./run.sh clean
```
